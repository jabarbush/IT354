Tutorial on Hoisting in Javascript

Hoisting:

Hoisting is a mechanism in Javascript that lets variable and named function declarations be brought to
to the top of the script and read/stored first. It only recognizes the declarations and does not bring
data values with them. The video above demonstrates an example of this.
