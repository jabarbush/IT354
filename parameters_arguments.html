Tutorial on Parameters and Arguments of Javascript

Optional Arguments:

These are arguments that you do not need because there is already a default value. However, if given a value, that will replace the default value. This is good for reducing errors!

function add(x, y = 1) {return x + y;}

Rest Parameter:

A rest parameter allows for any number of arguments! This is great if the number of arguments you need varies.

function add(...arguments){ }

Scope:

There is global scope and local scope. Global scope allows the function to see everything within a class. Essentially, you can see out, but you can't see in.
