Tutorial on Values and Types in Javascript

Values and Types:

Numbers- 3.123e+123
Numbers can be huge, but they have a limit. Know the limit (1.7976931348623157e+308). Once you go beyond this, you will only see "Infinity."

Infinity and -Infinity
These numbers are past Javascript's limits. They can still be manipulated!

NaN- isNaN(234) , isNaN("hello")
Not a number lets you know if something is not a number.

Strings- delete object.value;
Strings can go in ", ', or `. They hold as many characters as you might need. And each one can be indexed!
