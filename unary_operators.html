Tutorial on Unary Operators in Javascript

Unary Operators:

Increments/Decrements - i++ , i--
This is simply a way to add 1 or subtract 1 from a value.

Unary Plus/Minus - +3 , +false, -true
This is turns the vale into a number, if possible.

Logical Not - !true
This negates the value it prefaces.

Delete - delete object.value;
This will result in deleting data and the console will return true. It will also return true if the said data never existed.

Typeof - typeof 55 , typeof "hello"
This returns what kind of data type is in question.
