Tutorial on Functions of Javascript

Named Functions:

Named functions are just functions with names! You can use anonymous functions, but named functions allow for you to trace back errors easily.

function namedFunction(x,y){return x+y}

Function Expression:

Function expressions are functions stored into a variable. This is important because JavaScript uses hoisting, so regular function declarations are loading before the rest of the script, and you may not want that!

let varible = function add(x, y){return a+b}

Arrow Functions:

const add = (a, b) => a + b;
